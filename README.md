# 案卷实体治理与知识图谱检索系统

一个专业的案卷管理与知识图谱可视化系统，支持文档上传、实体抽取、知识图谱检索和数据可视化分析。

## 功能特性

### 5个核心页面

1. **文档上传与案卷列表页**
   - 支持拖拽上传 docx/pdf/png/jpg 格式文件
   - 归属单位和业务标签选择
   - 自动实体抽取
   - 案卷列表展示与排序
   - 快速跳转查看、图谱、时间轴

2. **全局检索结果页（双栏布局）**
   - 支持人名/单位/事件/案件编号/地点搜索
   - 多维度筛选（时间、单位、标签）
   - 左侧文字结果列表
   - 右侧知识图谱预览

3. **事件详情与时间轴页**
   - 案件基本信息卡片
   - 结构化案卷原文展示
   - 可交互时间轴
   - 所有关键信息可点击检索

4. **知识图谱下探页**
   - 全屏知识图谱可视化
   - 节点类型区分（人物/单位/事件/地点）
   - 支持缩放、拖拽、节点选择
   - 关联关系展示

5. **领导驾驶舱**
   - 核心数据统计卡片
   - 多维度可视化图表
   - 图表点击下钻
   - 快捷入口导航

### 5个通用弹窗

1. 文档上传进度弹窗
2. 实体抽取结果确认弹窗
3. 节点详情弹窗
4. 筛选弹窗
5. 批量操作弹窗

### 核心交互规则

- 所有人名、单位名、案件名、案件编号均可点击检索
- 所有图表支持点击下钻
- 图谱节点支持单击展开、双击跳转
- 列表行点击跳转详情页
- 全局检索入口互通

## 技术栈

- **前端**: 原生 HTML5 + CSS3 + JavaScript
- **图表**: 自定义 Canvas/SVG 渲染
- **图谱**: 力导向布局算法
- **样式**: CSS Variables + 深色主题

## 快速开始

1. 克隆仓库
```bash
git clone https://github.com/432ket/case-management-system.git
```

2. 打开项目
```bash
cd case-management-system
```

3. 启动项目
- 直接在浏览器中打开 `index.html`
- 或使用本地服务器（如 Live Server）

## 数据字段

### 基础字段
- 案件编号、案件名称、类型、标签、归属单位

### 时间/地点
- 案发时间、报案时间、地点

### 人员相关
- 人物列表（姓名+身份类型）、办理单位、办案民警

### 案件相关
- 案情摘要、处理结果
